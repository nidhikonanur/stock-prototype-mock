# Stock Prototype

## Local (dev)
- backend: `cd backend && npm install && node server.js`
- frontend: `cd frontend && npm install && npm start`

## Deploy
- Backend: Render (set MOCK=true for demo, or set MOCK=false and FINNHUB_KEY)
- Frontend: Vercel (Root directory: frontend; set REACT_APP_API_URL to backend URL)
